## About

Features:
- Small size (<1MB)
- Emoji panel
- Adjustable keyboard height for more screen space
- Number row
- Swipe space to move cursor
- Custom theme colors
- Minimal permissions
- Ads-free

Feature it doesn't have:
- GIFs
- Spell checker
- Swipe typing
